# ValMassoterapeuta

Site criativo para **massagem e depilação** com:
- agenda online (50 minutos por horário);
- seção de pagamentos;
- endereço e localização com mapa;
- gestão de sessões e valores;
- combos mensais de 1 a 10 sessões com desconto progressivo.

## Como executar localmente

Basta abrir o arquivo `index.html` no navegador.

## Estrutura

- `index.html` → estrutura da página
- `styles.css` → estilo (tema branco e rosa claro)
- `script.js` → lógica de agendamento, preços e combos
